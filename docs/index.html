Syllabus

Course Description

This course provides an introduction to practical web development for artists. Each student will learn the basics of HTML and CSS, and how to combine them into a functioning portfolio website. Students will learn how to work with rich media sources, create dynamic animations, and deploy content to the web using free and open‑source tools. The final project is to have a personal website up and running.

Time Commitment

  • Undergrad: 6 hrs/week (including lectures)
  • Grad: 8 hrs/week (including lectures)

Expected Expenses

Purchasing a domain name is optional but strongly encouraged (~$10/yr for .com, .me, .art, etc.).

Academic Integrity

Plagiarism—including uncredited code—is subject to disciplinary action per CalArts policy. If you use code written by someone else, include an attribution comment with the author's name and URL.

Accessibility

Students with documented disabilities should contact the Office of Student Affairs (A207) early in the semester to arrange accommodations.
